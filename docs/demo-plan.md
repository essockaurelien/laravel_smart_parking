# Demo Plan

1. Start Docker services and Laravel apps.
2. Seed parking spots and pricing.
3. Open the monitor page and show occupancy summary.
4. Register and login a user (Web UI).
5. Create a reservation (premium user) with card data.
6. Check-in and request charging (accept the quote).
7. Simulate charging progress updates.
8. Receive MQTT notification on completion (if enabled).
9. Check-out and show parking + charging payments.
10. Trigger the reservation expiry command (optional).
